# CoreBot
CoreBot is a minecraft companion bot using the "mineflayer", "mineflayer-navigate", "readline" and "fs" APIs and their dependencies. 
It can:
- Talk in chat,
- Use commands, 
- Follow players, 
- Attack players, 
- Display its health and food levels, 
- Manage its inventory.
